# ⚡ 408 & LeetCode：算法研修日志

> *"积跬步，以至千里。"*

## 📖 项目概述
本仓库是我备战 **2026 计算机考研 (408)** 以及日常算法训练的实录。
这不是一个精装修的代码库，而是一个记录逻辑推演与思维进化的**演练场**，主要包含：
* **数据结构**：深入理解树、链表、图论的底层实现。
* **双语训练**：使用 `C++` 构建核心逻辑，利用 `Python` 快速验证原型。
* **错题复盘**：在 `力扣本.xlsx` 中记录关键的解题陷阱与优化思路。

## 📅 进度时间轴 (2025)
| 阶段 | 核心考点 | 典型突破 |
| :--- | :--- | :--- |
| **1月** | 线性表基础 (Linear Lists) | `TwoSum`, `AddTwo`, `ReorderList` |
| **2月** | 树与递归 (Trees & Recursion) | `InorderTree`, `MaxDepth`, `SameTree` |
| **3月** | 图论与优化 (Graph & Optimization) | `FindPathInGraph`, `LRUCache`, `MinCostTree` |

## 🛠 仓库结构说明
```text
├── C_language/          # 底层实现尝试 (C)
├── Cplusplus/           # 基于 STL 的高效解法
├── *.ipynb              # 算法逻辑的可视化推导过程 (Python)
└── 力扣本.xlsx           # 🔴 错题集与复盘日志 (核心价值)
```
## 📝 关于代码风格的说明
为了模拟真实的 408 考场环境，本仓库中的大部分代码是在时间限制下编写的。 比起代码风格的优雅（PEP8/Google Style），我目前更关注：

算法正确性 (Correctness)

时空复杂度优化 (Time/Space Complexity)

手写代码的肌肉记忆
